# Flow 05. 전역 UI 및 공통 유틸리티

이 문서는 프로젝트 전반에 걸쳐 사용되는 전역 UI 시스템, 알림 방식, 그리고 TanStack Query의 전역 설정을 설명합니다.

## 1. Provider 계층 구조 (`src/app/layout.tsx`)

앱의 모든 전역 기능은 최상위 레이아웃에서 Provider들의 중첩으로 설정됩니다. 순서가 중요합니다.

```
<Providers>                   ← TanStack Query 클라이언트 주입
  <AuthProvider>              ← 앱 초기 세션 확인 및 전역 인증 상태 초기화
    <HeaderTitleProvider>     ← 동적 헤더 타이틀 공유
      {children}              ← 실제 페이지 콘텐츠
      <BottomNav />           ← 하단 네비게이션 바
      <GlobalLoginModal />    ← 전역 로그인 모달 (조건부 렌더링)
      <Toaster />             ← react-hot-toast 알림 출력 영역
    </HeaderTitleProvider>
  </AuthProvider>
</Providers>
```

## 2. TanStack Query 전역 설정 (`src/app/providers.tsx`)

`Providers` 컴포넌트는 `QueryClient`를 단 한 번 생성하고 앱 전체에 주입합니다.
전역 기본값을 통해 모든 쿼리에 일관된 동작을 부여합니다.

| 설정 | 값 | 의미 |
| :--- | :--- | :--- |
| `staleTime` | 60,000ms (1분) | 데이터를 가져온 후 1분간은 최신 데이터로 간주하여 재요청하지 않습니다. |
| `refetchOnWindowFocus` | `false` | 브라우저 탭을 다시 포커스해도 자동으로 재요청하지 않습니다. |
| `retry` | `1` | API 요청 실패 시 1번만 재시도합니다. |

> **개발 도구**: `<ReactQueryDevtools>`가 포함되어 있어 개발 환경에서 우측 하단 아이콘을 통해 캐시 상태를 실시간으로 검사할 수 있습니다.

## 3. 동적 헤더 타이틀 (`HeaderTitleContext`)

모바일 헤더의 타이틀은 페이지마다 다르게 표시되어야 합니다. `HeaderTitleContext`는 이를 위한 공유 채널 역할을 합니다.

*   **설정 (페이지 컴포넌트 내부)**: `useHeaderTitle().setCustomTitle('독서모임 설정')`
*   **읽기 (Header 컴포넌트 내부)**: `useHeaderTitle().customTitle`
*   **초기화**: 페이지를 떠날 때 `setCustomTitle(null)`을 호출하여 기본 타이틀로 되돌립니다.

## 4. 알림(Toast) 시스템 두 가지

프로젝트에는 목적에 따라 두 종류의 알림 시스템이 공존합니다.

### `react-hot-toast` (범용)
API 요청 성공/실패 등 일반적인 알림에 사용합니다. Mutation 훅 내부에서 주로 호출합니다.
```typescript
import toast from 'react-hot-toast';
toast.success('저장되었습니다.');
toast.error('오류가 발생했습니다.');
```

### `showCustomToast` (`src/utils/toastUtils.tsx`) (브랜드 알림)
체크모의 브랜드 색상과 Framer Motion 애니메이션이 적용된 커스텀 스타일 토스트입니다.
사용자에게 더 세련된 피드백이 필요한 경우 사용합니다. (화면 하단 중앙 팝업 형태)
```typescript
import { showCustomToast } from '@/utils/toastUtils';
showCustomToast('프로필이 업데이트되었습니다.');
```

## 5. 공통 유틸리티 (`src/utils/`)

| 파일 | 주요 역할 |
| :--- | :--- |
| `date.ts` | 날짜 포맷팅 (예: '2024-01-01' → '2024년 1월 1일') |
| `time.ts` | 시간 포맷팅 및 상대 시간 계산 (예: '3분 전') |
| `phone.ts` | 전화번호 형식 변환 (예: '01012345678' → '010-1234-5678') |
| `url.ts` | URL 파라미터 조작 및 경로 생성 유틸리티 |
| `notification.ts` | 알림 타입에 따른 이동 경로 매핑 유틸리티 |
| `groupMapper.ts` | 서버 응답 데이터를 UI 렌더링에 적합한 형태로 변환 |
